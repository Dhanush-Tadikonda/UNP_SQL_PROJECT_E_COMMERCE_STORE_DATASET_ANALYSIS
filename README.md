# UNP_SQL_PROJECT_E_COMMERCE_STORE_DATASET_ANALYSIS

The **Electronic Store Management System** is a database project developed using SQL to manage the operations of an electronic store efficiently. The main objective of this project is to design and implement a structured database that stores and organizes important business data such as product categories, products, suppliers, customers, orders, deliveries, returns, and inventory.

The project begins with the creation of a database named *D_Electronic_Store*, followed by the creation of multiple related tables. Each table is designed with appropriate primary keys and relationships to maintain data integrity. The system allows storing product details, tracking supplier transactions, managing customer orders, updating delivery status, and handling returns and exchanges.

Various SQL concepts are implemented in this project, including SELECT queries, WHERE conditions, comparison operators, AND/OR/NOT operators, LIKE operator, GROUP BY clause, and aggregate functions such as COUNT, SUM, and AVG. Different types of JOIN operations are used to combine data from multiple tables for meaningful analysis. ALTER and UPDATE statements are also used to modify table structures and data when required.

Overall, this project demonstrates how SQL can be applied in real-world business scenarios to manage data efficiently and support better decision-making.
